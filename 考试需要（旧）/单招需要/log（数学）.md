







$log_aN=b$ <=> $a^b = N$
- a 底数
- b 指数   对数
- N 幂     真数

- 常用对数：$log_{10}N$ ->  $lgN$
- 自然对数：$log_eN$  -> $lnN$

###### 例子：
lg1000 = 3
log_2√2 = 1/2
log_4??1?2.49bs <font style="color:red">1</font> = 0

#### 运算法则：
- $log_a(M*N)=log_aM+log_aN$
- $log_a(\frac{M}{N})=log_aM-log_aN$
- $log_a(M^N)=N*log_aM$  比如 $5^2⁺^2 =25^2$  $log_5 25^2 = 2*2$
