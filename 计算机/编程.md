## **一、常见命名约定**
### 1. **变量命名**
| 类型                     | 示例                | 适用场景                                             |
| ---------------------- | ----------------- | ------------------------------------------------ |
| **小驼峰 (camelCase)**    | `playerHealth`    | 局部变量、函数参数（Java/JS常用）                             |
| **下划线分隔 (snake_case)** | `player_health`   | C/C++/Python 常见                                  |
| **匈牙利命名法**             | `m_fPlayerHealth` | 旧时代 Windows 代码（`m_` 表示成员变量，`f` 表示 float）现在不常用了好像 |
| **前缀类型标记**             | `g_globalVar`     | `g_` 表示全局变量  现在不常用了好像                            |
| **常量全大写**              | `MAX_HEALTH`      | 常量、宏定义                                           |

### 2. **循环/临时变量**
| 用途               | 常见命名              | 说明                     |
|--------------------|----------------------|------------------------|
| **整数索引**        | `i`, `j`, `k`       | `for (int i = 0; ...)` |
| **范围循环变量**     | `item`, `element`   | `for (auto& item : list)` |
| **临时计算结果**     | `temp`, `tmp`       | `int tmp = a + b;`     |
| **迭代器**          | `it`, `iter`        | `auto it = vec.begin()` |

### 3. **函数命名**
| 风格               | 示例                  | 适用场景               |
|--------------------|----------------------|----------------------|
| **动词开头 (小驼峰)** | `getPlayerHealth()` | 获取数据               |
| **布尔值 (is/has/can)** | `isAlive()`        | 返回 `true/false`     |
| **事件回调**        | `onDamageTaken()`   | 游戏逻辑事件处理         |

## **三、代码示例对比**
### **1. 游戏开发示例（C++）**
```cpp
// 玩家类
class Player {
private:
    int m_health;       // 成员变量（匈牙利命名法）
    float m_moveSpeed;  

public:
    void TakeDamage(int damage) {  // 动词开头（小驼峰）
        m_health -= damage;
        if (IsDead()) {           // 布尔查询
            OnDeath();            // 事件回调
        }
    }

    bool IsDead() const { 
        return m_health <= 0; 
    }
};

// 临时变量 & 循环
void SpawnEnemies() {
    for (int i = 0; i < 10; ++i) {   // i 是索引
        Enemy& enemy = enemies[i];    // 引用类型
        enemy.SetPos(CalculateSpawnPos(i));
    }
}
```

### **2. 应用开发示例（Python）**
```python
# 用户服务类
class UserService:
    def __init__(self):
        self._db_connection = None  # 保护成员（单下划线）

    async def fetch_user_data(self, user_id: int) -> dict:
        """获取用户数据（snake_case）"""
        query = f"SELECT * FROM users WHERE id = {user_id}"
        result = await self._db_connection.execute(query)  # 异步调用
        return result
```
