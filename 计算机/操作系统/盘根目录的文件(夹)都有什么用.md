C盘的文件夹都是干啥的？
- $WinREAgent\$    安装升级系统用如果失败了也方便恢复
- hp   某品牌笔记本所以会有  里面是工具软件
- Intel   inter芯片组或集成显卡驱动的安装程序   有时还会有名为AMD或NIVDIA的文件夹（显卡）
- MSOCache   当Officials套件涉及修复或者更新时可能会用的这个文件夹
- perfLogs    性能日志
- Program Files   32位应用
-  Program Files (x86)   64位应用
-  ProgramData  应用程序的配置文件以及缓存文件  随着时间偏移会越来越大
- Recovery  用于恢复系统
- Windows
- 用户
