#菜单 #HTML #编程

<!--这是注释-->
HTML基础：
    标题：h1~h6 `<h1>标题名<\h1>`
    段落：`<p>段落<br><abbr title="内容">内容<abbr><\p>`
    [[链接]]：`<a href="https://www.runoob.com">菜鸟教程</a>`
    图像：`<img src="/images/logo.png" width="258".../>`
    视频：[这个比较麻烦](HTML视频.md)
HTML属性：
    class  为html元素定义一个或多个类名（classname）(类名从样式文件引入)
    id	定义元素的唯一id
    style	css代码
    script JavaScript 代码
    title	描述了元素的额外信息 (作为工具条使用)    
    value = "0"
    width = “20px”  宽
    onclick = "f()"  被点击时触发一个函数
    type = "text"
	align 位置
标签：
	ul（用于定义无序列表） 和 li （标记嵌套在ul中）[例子](一些例子.md#ul和li)
	ol（有序列表）
	-
	dl  用于定义盒子    
	dt   用于存放图片
	dd  用于存放内容
	-
	table（制表格）[[HTML制表格属性]]（不是css）
	tr（内）   valign （top middle bottom）（不是css）
	td（内内）align   colspan和rowspan单元格横跨列数
	-
	form  双标签  前后端交互有关 action服务器IP？method 传输协议？
	-
	input  输入（单标记）  [[HTML-input]]
	-
	textarea 文本框 cols 每行最多字符数量  rows行数
	-
	select 下拉菜单 size 可见选项数  multiple=multiple 多选功能
	option 下拉菜单的选项 selected=selected 默认选项
