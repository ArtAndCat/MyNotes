实现上的区别：
外：html文件和css文件分开
内：HTML中的head中的style中声明css
内联：直接在元素中声明

用在什么情况下？
外：当样式需要应用于很多页面时
内：单个文档需要特殊的样式时
内联：表现 内容混杂一起，会损失掉样式表的许多优势。
请慎用这种方法，例如当样式仅需要在一个元素上应用一次时。
要使用内联样式，你需要在相关的标签内使用样式（style）属性。Style 属性可以包含任何 CSS 属性。
