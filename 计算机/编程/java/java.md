格式：
```java
//这里写类  怎么这里写不了函数和变量（学C++的）
public class 文件名 {//Java文件中只能有一个public的类
    //在这里写函数和成员变量静态变量
    public static void main(String[] args) {//主函数
        //这里写局部变量和逻辑什么的
        System.out.println("Hello World"); // 输出Hello World
    }
}
```
