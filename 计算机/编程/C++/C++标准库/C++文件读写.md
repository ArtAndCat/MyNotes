文件读写
```cpp
#include <fstream>//读写文件
```
创建读写器（前面+o为只读i为只写）
```cpp
fstream 读写器名 ;  //读写
```
指定读写器要打开的文件
```cpp
读写器名.open("路径",打开方式);
```
打开方式：（提示：与|配合使用  不是||）==前面得加 std::ios_base::___==
1. in  读
2. out  写
3. ate  结尾
4. app  追加
5. trunc   如有先删
6. binary  二进制
存入数据
```cpp
读写器名 << __;
```
错误
```cpp
读写器名.cidbek;//之前记的关闭，实际的话好像不对
```
判断是否打开（返回布尔型）
```cpp
读写器名.is_open();
```
#### 关闭
```cpp
读写器名.close();//关闭
```

#### 读写的4个方法
行读写 1   这样就行？？？（读取到结尾时自动返回假）
```cpp
while (数据名 << ifs){}
```
行读写 2   提示：max字节数可以用  [[sizeof（）]]函数
```cpp
读写器名.getline(数据名,max字节数);    // 行读取
```
行读写 3  
```cpp
getline(读写器名,数据类型);    // 这里可以用string
```
行读写 4 （不推荐）
```cpp
读写器名.get(char &)    // 每次只返回一个字符
```
