动态数组，可以在运行时调整大小。它提供了类似于数组的随机访问功能，并且支持在尾部高效地插入和删除元素。vector容器的元素在内存中是连续存储的，因此可以通过索引快速访问元素。它适用于需要频繁对元素进行随机访问的场景。

动态扩展的原理：先申请n个内存空间当空间被占满时寻找新的内存空间然后复制原来的到新的内存空间释放旧的内存空间

头文件：
```cpp
#include <vector>
```
#### 创建容器：
```cpp
vector <数据类型> 容器名;
```
或者:
```cpp
vector(v.begin(), v.end());//将v[begin(), end())区间中的元素拷贝给本身。
vector(n, elem);    //构造函数将n个elem拷贝给本身。
vector(const vector &vec);         //拷贝构造函数。
```

#### 赋值操作：
```cpp
// 直接=号 懒得记了
容器名.assign(beg,end);  // 将[beg, end)区间中的数据拷贝赋值给本身。
容器名.assign(n, elem);  // 将n个elem拷贝赋值给本身。
```

注意事项（菜鸟教程）
1. 容器名.push_back( ) 成员函数在向量的末尾插入值，如果有必要会扩展向量的大小。
2. 容器名.size( ) 函数显示向量的大小。(单位：字节)
3. 容器名.begin( ) 函数返回一个指向向量开头的迭代器。
4. 容器名.end( ) 函数返回一个指向向量末尾的迭代器。


自定义数据类型
声明结构体：
```cpp
class p {
public:
	int a;
	p(){
		a = 10;
	}
}
```
创建容器
```cpp
vector <p> aa;
```
向容器添加东西
```cpp
p p1;
aa.push_back(p1);
p p2;
aa.push_back(p2);
```

```cpp
vector<p>::iterator aaa = aa.begin();
aaa++;//这是一个指针
aaa->a++
```
