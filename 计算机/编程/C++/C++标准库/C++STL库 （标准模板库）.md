## STL六大组件，分别是:[容器](STL容器.md)、[算法](STL算法.md)、[迭代器](STL迭代器.md)、[仿函数](STL仿函数.md)、适配器（配接器）、空间配置器
1. [容器](STL容器.md)：各种数据结构，如vector、list、deque、set、map等,用来存放数据。
2. [算法](STL算法.md)：各种常用的算法，如sort、find、copy、for_each等
3. [迭代器](STL迭代器.md)：扮演了容器与算法之间的胶合剂。
4. [仿函数](STL仿函数.md)：行为类似函数，可作为算法的某种策略。
5. <font style="color:rgb(25,25,250);">适配器</font>：一种用来修饰容器或者仿函数或迭代器接口的东西。（实际运用较少，不讲）
6. <font style="color:rgb(25,25,250);"> 空间配置器</font>：负责空间的配置与管理。（这个也不讲）
